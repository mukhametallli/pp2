# 🎨 Paint App (Pygame)

## 📌 Overview
A simple drawing application built with Pygame that allows users to draw shapes, use colors, erase, and create sketches on a canvas.

---

## ✏️ Features
- 🖌️ Brush tool (free drawing)
- ▭ Rectangle tool
- ⚪ Circle tool
- 🧽 Eraser tool
- 🎨 Color palette
- 📏 Adjustable brush size
- 🧹 Clear canvas option
- 🧰 Toolbar interface

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| B | Brush tool |
| R | Rectangle tool |
| C | Circle tool |
| E | Eraser tool |
| + | Increase brush size |
| - | Decrease brush size |
| ESC | Exit app |

---

- Python 3.x
- Pygame
