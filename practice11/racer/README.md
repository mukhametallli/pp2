# 🚗 Car Racer Game (Pygame)

## 📌 Overview
A simple 2D car racing game built with Pygame. The player controls a car, avoids enemy vehicles, and collects coins while trying to survive as long as possible.

---

## 🎮 Features
- 🚗 Player-controlled car (left/right movement)
- 🚙 Enemy cars with increasing difficulty
- 🪙 Coin collection system
- 📊 Score system (based on survival)
- 🎵 Background music
- 💥 Crash and coin sound effects
- 💀 Game Over screen with restart option

---

## 🕹️ Controls
| Key | Action |
|-----|--------|
| ← | Move Left |
| → | Move Right |
| R | Restart game |
| Q | Quit game |

---

## ⚙️ Requirements
- Python 3.x
- Pygame
