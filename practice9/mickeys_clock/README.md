# 🎵 Keyboard-Controlled Music Player

A lightweight music player that allows users to manage a playlist entirely through keyboard shortcuts.

## Features
- **Playlist Support**: Loads multiple audio files from the `/music` directory.
- **Visual Feedback**: Displays the current track name and playback status.
- **Mixer Integration**: Uses `pygame.mixer` for high-quality audio playback.

## Keyboard Controls
| Key | Action |
| :--- | :--- |
| **P** | Play Current Track |
| **S** | Stop Music |
| **N** | Next Track |
| **B** | Previous Track (Back) |
| **Q** | Quit Application |

## Requirements
- Python 3.x
- Pygame
- Music files in `.mp3` or `.wav` format within the `/music` folder.
